# 導入手順

## .batファイル経由
1. dl_develop_common.batをダウンロード
2. プロジェクトのAssetsフォルダ内に"dl_develop_common.bat"配置し実行
3. 追加された"_develop_common/Package_Json" にある "overwrite_package.bat" を実行
4. [UniRx](https://assetstore.unity.com/packages/tools/integration/unirx-reactive-extensions-for-unity-17276?locale=ja-JP&srsltid=AfmBOoqdP03q7gNgMR-Vy3dlW6R-VnqcC9KtcWgvu9iBmamkAR0NdLX4) をPackageManagerからインポート（Unity Asset Store からの追加が必要）

## githubページ経由
1. githubページから"_develop_common" をダウンロード
2. プロジェクトのAssetsフォルダ内に "_develop_common" 配置
3. "_develop_common/Package_Json" フォルダにある "overwrite_package.bat" を実行
4. [UniRx](https://assetstore.unity.com/packages/tools/integration/unirx-reactive-extensions-for-unity-17276?locale=ja-JP&srsltid=AfmBOoqdP03q7gNgMR-Vy3dlW6R-VnqcC9KtcWgvu9iBmamkAR0NdLX4) をPackageManagerからインポート（Unity Asset Store からの追加が必要）

# 更新方法
1. git_pull.batを実行する ※ common_develop内の作業は**すべて取り消されます**
